# Brain Challenge Arcade

Brain Challenge Arcade is a web-based mini-game project built with HTML, CSS, and JavaScript.

## Features

- Story Mode with chapters
- Image Guess game
- Memory Test game
- Card Matching game
- Color Bubble Challenge
- Leaderboard
- Achievements
- Profile page
- Settings page
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript
- LocalStorage

## How to Run

Open `index.html` in a web browser.

For the best experience, use VS Code Live Server.
